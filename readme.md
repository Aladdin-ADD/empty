# node
> a Node.js boilerplate repo, with some useful deps.

## Usage

```bash
$ npm install -g degit
$ degit tplss/node
```

## Features

+ eslint (eslint-config-egg + eslint-plugin-node)
+ husky
+ rollup
+ mocha
